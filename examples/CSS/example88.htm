Sir Timothy Berners-Lee

Sir Timothy Berners-Lee, OM, KBE, FRS, FREng, FRSA, born 8 June 1955, is a British engineer and computer scientist and MIT professor credited with inventing the World Wide Web, making the first proposal for it in March 1989. On 25 December 1990, with the help of Robert Cailliau and a young student at CERN, he implemented the first successful communication between an HTTP client and server via the Internet.
